Remarks

This static class governs mortality within the Simulation.

Based on age-specific probabilities stored in MortalityTuples, the Reaper evaluates whether a human should pass by ThrowDiceOfLife(in Human), which simulates the uncertainty of life. After a date is selected, the Reaper checks IsThisTheDay(Human, in DateTime), which simply checks whether death is scheduled. If so, Yoink(Human, in DateTime) pulls the soul from the world and return it to the Steppe.

No negotiation. No mercy. Just one final tick.
