Remarks

This static class governs mortality within the simulation.

Based on age-specific probabilities stored in MortalityTuples, the Reaper evaluates whether a human should pass. This can be triggered by ThrowDiceOfLife, which simulates the uncertainty of life, or IsThisTheDay, which simply checks whether death is scheduled. If so, Yoink pulls the soul from the world.

No negotiation. No mercy. Just one final tick.
